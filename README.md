# Elementary-CA
Elementary Cellular Automata 1D


## Installation
```
$ git clone https://github.com/Coac/elementary-CA
$ cd elementary-CA
$ npm install
```

## Development
```
$ npm run dev
$ open http://localhost:8080/
```
Running the webpack-dev-server.
